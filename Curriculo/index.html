  • Controle de Estoque com CustomTkinter

    Python | CustomTkinter | PostgreSQL

    Este projeto é um sistema de controle de estoque com interface gráfica desenvolvida em CustomTkinter, utilizando o banco de dados PostgreSQL para armazenar e gerenciar as informações. O objetivo é fornecer uma solução simples e eficiente para controle de inventário, com uma interface amigável e fácil de usar. O sistema está em estágio inicial, mas já permite a visualização, adição e remoção de itens do estoque.

  • Downloader de Vídeos do YouTube

    Python | CustomTkinter | yt-dlp

    Este projeto consiste em um aplicativo para download de vídeos do YouTube, desenvolvido em Python com uma interface gráfica usando CustomTkinter. O objetivo é proporcionar uma solução simples e acessível para usuários que desejam salvar vídeos localmente, com funcionalidades que permitem selecionar a qualidade e o formato do vídeo a ser baixado. O projeto está em fase inicial de desenvolvimento, mas já apresenta uma interface intuitiva e funcional.
